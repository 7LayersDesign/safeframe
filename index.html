How to use SafeFrames

SafeFrame provides a consistent JavaScript API for safely including 3rd party web content into your web pages, while also allowing access to a controlled set of rich interaction options not traditionally available in iFrames.

This project helps websites and advertisers implement SafeFrame.

Overview

The IAB's SafeFrame specification has been developed with the aim of improving the way display advertising appears.

If you want to read more about this check out

  • An introduction to SafeFrame http://www.iab.net/safeframe
  • An infographic the IAB produced on why iframes are so dated http://www.iab.net/safeframe/safeframe_infographic
  • Original Source code on Sourceforge http://sourceforge.net/projects/safeframes/
  • A straight fork of this code on Github https://github.com/safeframe/safeframe

About this Project

There's a lot we like about the SafeFrame initiative. With this project we are taking the examples that have been provided in the project and making them (hopefully) a little easier to understand how they work and how to implement.

This project is working alongside our main project, Wanda, which is a standardized Javascript library that makes ads look and work better.
